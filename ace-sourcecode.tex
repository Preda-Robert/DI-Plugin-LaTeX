\section{Building and running the extension}

Developers load the repository in VS Code, run \texttt{npm install}, then \texttt{npm run compile}. Pressing \texttt{F5} launches an Extension Development Host with the plugin loaded. Sample folders under \texttt{sample/} should be opened as workspace roots when reproducing the manual tests in Chapter~\ref{chap5:evaluation}.

\section{Sample-generated artifacts}

Generated files are written to \texttt{GeneratedDI/} within each sample when \textbf{DI: Generate builders/factories files (workspace)} is executed. Committed copies in the repository illustrate expected generator output; regenerating after plugin updates is recommended before comparing against thesis tables.
